# Storage Policy — ADANiD OpenCloud

## Public Assets
- Code, specs, samples → **GitHub**
- Datasets → **Kaggle**
- Models → **Hugging Face**

## Private/Educational Data
- Full QuranLab audio → **Gated Kaggle Dataset** (requires approval)
- Biometric templates → **Local encrypted vault only** (never cloud)

## No External Cloud Dependencies
- ❌ No Firebase
- ❌ No AWS/Azure/GCP for core data
- ✅ Only open, auditable platforms: GitHub, Kaggle, Hugging Face
