# QuranLab AI — قرآنی تلاوت کا تجزیاتی نظام

## نظرة عامة
QuranLab AI ایک **تعلیمی و روحانی مقاصد** کے لیے بنایا گیا **Open-Core Islamic AI** سسٹم ہے جو:
- قرآنی تلاوت کا تجزیہ کرتا ہے  
- تجوید کے اصولوں کی خلاف ورزی کا پتہ لگاتا ہے  
- تلفظ کی درستگی کو سکور دیتا ہے
