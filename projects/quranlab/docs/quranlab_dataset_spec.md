# QuranLab Dataset Specification v1.0

## Purpose
Enable AI-based Tajweed error detection and pronunciation scoring for Quranic recitation.

## Data Sources
- Audio recorded under supervision of certified Qaris
- Labels verified by Tajweed scholars
- Qira'at: Hafs 'an 'Asim (primary), Warsh (secondary)

## File Structure
- `metadata.csv`: Central index
- Audio: MP3 format, 16kHz, mono
- Labels: JSON with time-aligned Tajweed rules

## Hosting
- **Public samples**: This GitHub repo
- **Full dataset**: Kaggle (gated access for educational use)
